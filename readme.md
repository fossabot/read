## Read
Example Atomic Design with DDD (Domain Driven Design)

## Requirement
  - [node.js](http://nodejs.org/)
  - [yarn](https://yarnpkg.com/en/)

## Quick Start

```bash
# clone repository
$ git clone https://github.com/logustra/read.git

# open folder read
$ cd read

# instal dependencies
$ yarn install

# copy file .env.example to .env
$ cp .env.example .env

# build and serve with express
$ yarn start
```
